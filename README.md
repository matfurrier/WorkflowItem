## üíº App Rules and Flows
The efficiency of our Workflow APP is based on a few strategic rules and flows. Here are the main rules that guide the use of the app:

- When registering a new item, it must be sent for validation by the involved departments.
- Collaborators can only edit the fields related to their respective departments.
- Items can only be edited as long as the "VALIDATED" option is not marked. Once this option is marked, the tab fields will be disabled and no further edits can be made.
- After the item is validated by all departments, the item registration status will be changed to "APPROVED".
